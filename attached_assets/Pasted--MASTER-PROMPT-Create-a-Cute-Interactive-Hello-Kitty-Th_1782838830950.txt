# MASTER PROMPT: Create a Cute Interactive Hello Kitty Themed Long-Distance Relationship Website

Create a beautiful, emotional, and interactive long-distance relationship website using **Next.js 15, React, Tailwind CSS, GSAP, ScrollTrigger, Lenis Smooth Scroll, and Framer Motion**.

The website should tell our love story through a **scroll-driven interactive experience** featuring smooth animations, parallax effects, cute transitions, scrapbook aesthetics, and a cozy Hello Kitty inspired theme (inspired by the aesthetic only, not direct copyrighted assets).

The overall atmosphere should feel:

* Warm
* Cozy
* Romantic
* Cute
* Dreamy
* Nostalgic
* Emotional
* Soft
* Magical

The website should feel like a combination of:

* An interactive scrapbook
* A digital love letter
* A memory diary
* A cute animated storybook
