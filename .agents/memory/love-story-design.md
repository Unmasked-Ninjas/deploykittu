## Palette
- Midnight ink: #0A0816 (main bg)
- Deep plum: #1A0A2E (section bg, card bg)
- Champagne gold: #C9A96E (primary accent)
- Dusty rose: #B8637A (love accent)
- Warm cream: #F0E6D6 (primary text)

## Fonts
- Cormorant Garamond — headings, section titles (italic for romance)
- Montserrat 300-400 — body, UI, nav
- Pinyon Script — love notes, letters, intimate text

## Key utility classes (index.css)
- .glass-dark — dark glass card (backdrop-blur, gold border)
- .gold-glow — gold text shadow
- .rose-glow — rose text shadow
- .shimmer-text — animated gold gradient text
- .font-display / .font-script / .font-body

**Why:** Replaced baby-pink kawaii look per user request; cinematic dark romance aesthetic.
